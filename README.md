# api-test-framework
API Test Framework for testing https://helloacm.com/tools/unix-timestamp-converter/

## Installation 

1. Install (node 18)[https://nodejs.org/dist/v18.14.2/node-v18.14.2-x64.msi]
2. Go to API Test Framework root directory 
3. Run command `npm install`

## Run tests

There are two ways of running tests
1. `npm run api-tests` gui
2. `npm run api-headless` cli
